### Запуск контейнеров
```sh
docker-compose up -d
```

### Остановка и удаление контейнеров
```sh
docker-compose down
```

### Список запущенных контейнеров
```sh
docker-compose ps
```

### Перезапуск контейнеров
```sh
docker-compose restart
```

### Остановка контейнеров без удаления
```sh
docker-compose stop
```

### Запуск остановленных контейнеров
```sh
docker-compose start
```
